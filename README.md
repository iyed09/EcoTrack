# EcoTrack – Suivi de consommation et gestion des déchets

EcoTrack est une application web qui permet de gérer les produits, leurs déchets et la consommation afin de suivre l’impact environnemental.  
Ce projet est réalisé dans le cadre du module **Projet Technologies Web 2A**.

## Description

EcoTrack a pour objectif de :
- Gérer les **produits** (création, affichage, modification, suppression).
- Gérer les **déchets** associés aux produits.
- Suivre la **consommation** (énergie, ressources, etc.).
- Organiser le projet selon une architecture **MVC en PHP**.
